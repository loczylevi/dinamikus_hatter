Ervin atyánkról

Szent atyánk rendünk vezetője feltehetőleg a görög istenek Zeusz és Héra szerelem gyermeke. Születésekor egy python random szám generátor döntött arról hogy hány évig éljen szeretett tanítónk. Csatoltunk is egy példát ezzel is felizgatva vizuális ingercsokorunkat.

import random
veletlen = ramdom.randit(1,100000)
>>> veletlen

Mint a fent említett példában látható az ősmagyar python script egy igen szerencsés életkort szabott ki császárunknak.
